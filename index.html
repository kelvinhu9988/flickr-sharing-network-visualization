Flickr Sharing Network Visualization

Photo Similarity Network

This section tries to visualize the similarity network of a chosen photo. An edge between photo nodes is formed when two photos either are uploaded by the same user or share same tags. Grey links signify same tags. Brown links signify being uploaded by the same user. Red links signify same tags as well as being uploaded by the same user. User can mouse hover a photo node to see more information about that photo.




User Interaction Visualization

This section focuses on visualizing a user's interaction with other users. The interaction on Flickr mainly includes tagging and commenting.

• Who is commenting on your photo?

The network shows the users who have commented on the photo(s) that the central user posted. The radius of a user node signifies the total number of comments that the user has received, which can be regarded as the popularity of that user. The length of the edge connecting two user nodes signifies the time duration between the non-central user's comment date and the current date. The shorter the edge is, the most recent the commenting activity between two users was. Since the data is outdated, the current date is assumed to be the most current comment date among all the non-central users' comment dates.




User Sharing Habit Visualization

This section focuses on visualizing user behavior that can reflect a user's sharing habit.

• What types of photos are you sharing?

The radial chart shows the most common (at most 8) photo types that a user has posted. The labels along the outer circle are the photo types. The vertical axis encodes the number of times a particular kind of photo has been posted. The arc segments are sorted in descending order. User can click on the "Update" button to view the sharing habit of another user in the data.
